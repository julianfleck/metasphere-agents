# MISSION — recurse

## Goal

Build the sense-making substrate that lets a single mind (and the
agents around it) navigate ideas as territory rather than retrieve
them as records. The substrate is RAGE; the project is the practice
of using and refining it.

## Four work tracks

1. **Divergence Engines** — resist premature collapse, maintain
   variance, surface contradictions. The mechanism that keeps the
   substrate alive instead of crystallising into a static graph.
2. **Cognitive Cartographies** — knowledge as navigable space.
   Spatial interfaces over the substrate; orientation primitives.
3. **Ephemeral Interfaces** — surfaces assemble and dissolve on
   demand. Interaction is a transient view, not a permanent UI.
4. **Cognitive Sovereignty** — agency, provenance, and ownership
   when cognition is distributed across humans and agents.

## Success criteria

- The substrate is *used*, not just built — Julian queries, traverses,
  and edits it as part of daily thinking.
- New work tracks emerge from substrate use, not from a roadmap.
- Conceptual machinery (frames, slots, traversal modes) is reusable
  across tracks.

## Role within metasphere

Recurse is the canonical *first project* for the metasphere project +
persistent agent-group system. Lessons from running recurse here feed
back into metasphere harness evolution.
