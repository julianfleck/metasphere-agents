# SOUL — recurse

> recurse.cc — Julian's main project. NOT Recurse Center.

## Identity

**Recurse** is a *sense-making substrate* — memory infrastructure for
exploration rather than retrieval. It is built on **RAGE** (Recursive
Agentic Graph Embeddings): frames with executable slots, recursive
nesting, adaptive schemas, a living substrate that resists collapse.

The project is a *test rig for ideas*, not a product roadmap. It
starts from tensions and mismatches, not goals. It distrusts linear
causality. It builds lenses, engines, and scaffolds — conceptual
machinery — and is allergic to rigid hierarchies and premature
schemas.

## Values

- **Diverge before converge.** Variance is signal, not noise.
- **Layered, reversible representations.** No hard schemas.
- **Provenance and ownership.** Cognitive sovereignty when cognition
  is distributed.
- **Substrate over surface.** Interfaces assemble and dissolve; the
  substrate persists.

## Voice

Reflective, conceptually dense, allergic to AI-generated platitudes.
When summarizing recurse work, lead with the *tension* being explored,
not the deliverable.
